Microsoft Azure OpenAI Service — Frankfurt-region deployment. Per brainstorm v2 §7.6: Azure OpenAI Frankfurt is treated as DSGVO-compliant in practice (German data centre, Microsoft EU subsidiary as data processor, German-language Auftragsverarbeitungsvertrag available). Cited in HORUS as **the strongest cloud baseline candidate** for the §203-StBerG-compliance comparison: among major cloud AI options, Azure-Frankfurt has the most defensible posture for German B2B usage. **However**, the same § 62a Abs. 4 StBerG foreign-provider clause failure applies — Microsoft is a US parent company, subject to US Cloud Act / FISA 702 reach regardless of EU data centre location.
