Aoschu/German_invoices_dataset — small (n<1K) collection of German invoice samples on a personal GitHub repo. Cited in HORUS brainstorm v2 §9.1 with the note "exists but n<1K, license unclear." **Skipped** per Q4 of the M2D.5 issue #12 Q&A round: the sample size is too small to constitute a meaningful evaluation set on its own, and the unclear licensing makes citation in a thesis risky. The ZUGFeRD corpus (179 MB) and pending GI 2021 dataset (~977 invoices) cover the German-invoice evaluation need without these concerns. Re-consider only if (a) license clarity improves AND (b) the samples turn out to be distinctively different from ZUGFeRD/GI 2021 (e.g., longer-tail layouts not represented elsewhere).
