OmniDocBench — comprehensive document-understanding benchmark from OpenDataLab on Hugging Face. Modern (2024+) benchmark covering multiple document types, languages, and extraction tasks. Cited in HORUS as a **modern multi-domain benchmark** (per brainstorm v2 §15 Datasets); the most current of the listed datasets. Useful for: (a) calibrating Layer-1 VLM performance on the same benchmark used in current Oct-2025 VLM-cohort papers (Granite-Docling, olmOCR-2, etc., likely report numbers on this); (b) cross-language generalization checks if the German-invoice scope ever needs international comparison. Primary evaluation remains German B2B invoices via ZUGFeRD; OmniDocBench is the cross-cutting comparison axis.
