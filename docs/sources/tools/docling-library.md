Docling open-source library (Apache 2.0). Customizable ensemble pipeline (parse + OCR + layout/tables + optional VLM). Self-hostable, MLX-capable. Now part of Linux Foundation's Agentic AI Foundation. Cited in HORUS as the **orchestrated-specialists Layer-1 baseline** to be benchmarked against single-shot VLMs like Granite-Docling (the integrated VLM mode within Docling) and olmOCR-2 (a separate single-shot VLM run independently of Docling). The Docling vs Granite-Docling comparison sits inside Layer 1 — orchestrated vs single-shot architecture — and is independent of the Layer-1 VLM-cohort comparison.
