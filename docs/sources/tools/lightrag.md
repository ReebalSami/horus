LightRAG — open-source hybrid retrieval framework combining vector embeddings with knowledge-graph reasoning. Designed as a lighter-weight, hybrid-aware alternative to Microsoft's full GraphRAG. Cited in HORUS as the **primary Layer-3 candidate** (per brainstorm v2 §7.3): given Han et al. 2025's finding that pure GraphRAG often loses to vanilla vector RAG, LightRAG's hybrid posture is the most defensible Layer-3 architecture for the thesis. Primary research question shifts to "*when* does the graph help, *when* does it hurt, can we predict per query" — LightRAG provides the framework to answer that.
