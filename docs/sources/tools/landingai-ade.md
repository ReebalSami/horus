Landing AI ADE (Agentic Document Extraction). Three deployment modes plus Zero Data Retention (ZDR): (1) SaaS in US AWS Ohio or EU AWS Ireland; (2) VPC containerized in customer's Snowflake / AWS / Azure / GCP; (3) fully on-premise installation. Security overview at https://landing.ai/security-at-landingai. Cited in HORUS as a **commercial cloud-baseline reference** for the §203-StBerG-compliance comparison: the honest cloud-vs-local comparison is "EU-hosted ZDR-enabled cloud SaaS vs local open-source," not "any cloud vs any local" (per brainstorm v2 §7.7). LandingAI ADE belongs in the cloud-baseline column because its EU + ZDR + VPC posture is closer to legal compliance than generic cloud APIs, while still failing the §62a-StBerG foreign-provider clause for full Steuerberater compliance.
