HippoRAG (and successor HippoRAG2) — RAG architecture inspired by hippocampal indexing theory; designed for multi-hop reasoning where the question requires traversing multiple knowledge units. Open-source, OSU NLP Group. Cited in HORUS as a **Layer-3 alternative** to LightRAG and Microsoft GraphRAG (per brainstorm v2 §7.3). Particularly relevant if the thesis's German-tax-domain queries turn out to be multi-hop heavy (e.g., "what's the total VAT charged across all invoices from Vendor X in Q4?"); single-hop queries favour vanilla vector RAG instead.
