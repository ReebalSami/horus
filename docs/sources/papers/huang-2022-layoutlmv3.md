LayoutLMv3 (Microsoft Research) — multimodal pre-trained transformer combining text + image + layout signals via unified masking objectives. ACL 2022. Established the multi-stage extraction-then-understanding pipeline that dominated 2022–2024 document AI. Cited in HORUS as the **historical reference** representing the orchestrated-pipeline approach (OCR + layout + LM) — the alternative to the OCR-free single-shot VLMs that the thesis evaluates against (per brainstorm v2 §7.1 + §7.8).
