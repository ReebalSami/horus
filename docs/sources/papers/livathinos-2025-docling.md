Docling technical report (IBM Research). Documents the customizable open-source ensemble pipeline for converting PDFs and images to structured representations. Combines parse + OCR + layout/tables + optional VLM stages, all configurable. Now part of Linux Foundation's Agentic AI Foundation. Granite-Docling is the single-pass VLM alternative within the Docling ecosystem (see `docs/sources/papers/ibm-2025-granite-docling.md`). Cited in HORUS as the **orchestrated-specialists baseline pipeline** to be benchmarked against single-shot VLMs at the experiment phase (per brainstorm v2 §7.8). The Docling-vs-Granite-Docling comparison sits inside Layer 1 — orchestrated vs single-shot — independently of the Layer 1 VLM-cohort comparison.
